# 🔐 Incident Response Lab – NIST Framework



## 📌 Overview

This project demonstrates a hands-on simulation of an incident response process based on the NIST framework.



## 🎯 Objective

Apply the 4 phases of incident response:

1. Preparation  

2. Detection & Analysis  

3. Containment, Eradication & Recovery  

4. Post-Incident Activity  



## 🧪 Scenario

Phishing email leads to malware infection on a user machine.



## 🔥 What I Learned

- Structured incident response workflow

- Importance of logs and evidence

- Containment must be fast and decisive

- Documentation is critical in cybersecurity
